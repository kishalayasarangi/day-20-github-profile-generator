# Day 20 — GitHub Profile README Generator

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A tool that generates a beautiful GitHub profile README
by filling in a simple form.

## How to view
Open `index.html` in your browser.

## Features
- Fill form and get instant README preview
- Copy generated markdown with one click
- Includes badges, stats, skills section
- Live markdown preview
